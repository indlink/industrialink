Installation Guide

Up and running in 5 minutes

01
Download the installer
Click the download button above to get IndustriaLink-Setup-1.0.0.exe from GitHub Releases. Verify the SHA-256 checksum if needed.
02
Run the setup wizard
Right-click the installer and select "Run as administrator". The wizard will guide you through the install path and create a desktop shortcut automatically.
Windows SmartScreen may show a warning on first run. Click "More info → Run anyway" — the binary is unsigned for now.
03
Configure your SQL Server connection
Open config/config.json in the install directory and fill in your SQL Server host, database name, and credentials.
04
Launch IndustriaLink
Double-click the desktop shortcut. The PyQt6 launcher boots the server and opens the platform in your browser at http://localhost:5000. Database tables are created automatically on first run.
05
Connect your first PLC or device
Log in as Admin, go to Settings → PLC Connections, and add your Siemens S7-1200 (port 102), Modbus TCP device (port 502), or MQTT broker. The live monitor will populate immediately after the first poll cycle.
For Siemens S7-1200: disable "Optimized block access" on your DB in TIA Portal and enable PUT/GET communication before connecting.

New Desktop application packaging with PyQt6 + PyInstaller + Inno Setup
Platform converted from a web app to a Windows desktop application. The PyQt6 launcher boots Flask/Werkzeug in the background, redirects all server logs into the launcher window, and eliminates the console window entirely.
New ML Dataset Validation — CWRU bearing fault classification
New vibration analysis tab with drag-and-drop CSV upload, tri-axial feature extraction, RandomForest classifier (200 estimators, balanced class weights), file-level GroupKFold CV to prevent data leakage, and RUL estimation. PDF/Excel export via jsPDF and SheetJS.
New Prescriptive maintenance engine with multi-plan scoring
Five ranked maintenance plans generated per asset using fault-profile tables, global Pareto normalization, safety-critical override logic, and dynamic risk reduction from failure_prob and health_score. Confidence flag for tied scores.
New GMAO parts request & approval workflow
Technicians submit parts requests with urgency levels via the Field App. Admins approve or reject from a unified panel in gmao.html. Replaces direct inventory deduction with a full audit trail and badge counts.
Fix SQL Server T-SQL dialect — full SQLite syntax removal
Replaced all SQLite-specific syntax (AUTOINCREMENT, strftime, PRAGMA) with T-SQL equivalents (IDENTITY, CONVERT, sys.columns) across the entire codebase for full SQL Server 2017+ compatibility.
Fix Siemens S7-1200 snap7 connection — optimized block access
Diagnosed and documented the TIA Portal "Optimized block access" issue that prevented snap7 from reading DB variables by absolute byte offset. Added setup guide for disabling this and enabling PUT/GET communication.
Improve Thread-safety and connection pool hardening
Added _driver_lock across all PLC driver access paths, fixed non-atomic cooldown/pump activation, resolved connection leaks in kpi_iiot.py and five other API endpoints, and corrected background threads calling conn.close() instead of returning to pool.
